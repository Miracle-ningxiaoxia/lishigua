# Audio Files

Please add your ambient background music file here:

- **File name**: `ambient-bgm.mp3`
- **Recommended format**: MP3 or OGG
- **Duration**: 2-5 minutes (will loop automatically)
- **Volume**: Mixed to -12dB to -18dB for comfortable listening
- **Style suggestions**: 
  - Lo-fi ambient
  - Cinematic soundtrack
  - Piano/acoustic guitar
  - Minimal electronic

The audio player will automatically fade in/out with a 1.5-second transition for a premium listening experience.
