# Gallery Images

请按照以下规格准备您的画廊图片：

## 图片命名规范

### 缩略图 (Thumbnails)
用于轮播展示，文件较小以提升加载速度：
- `thumb-1.jpg`
- `thumb-2.jpg`
- `thumb-3.jpg`
- `thumb-4.jpg`
- `thumb-5.jpg`
- `thumb-6.jpg`

**推荐规格**：
- 尺寸：800x1000px（4:5 比例）
- 格式：JPEG
- 质量：80%
- 文件大小：< 200KB

### 高清大图 (Full Images)
用于 Lightbox 全屏查看：
- `full-1.jpg`
- `full-2.jpg`
- `full-3.jpg`
- `full-4.jpg`
- `full-5.jpg`
- `full-6.jpg`

**推荐规格**：
- 尺寸：2400x1800px（4:3 比例）
- 格式：JPEG
- 质量：90%
- 文件大小：< 1MB

## 图片内容建议

1. **城市夜景** - 高楼大厦、灯火通明的街道
2. **樱花树下** - 春天的粉色花海
3. **公路旅行** - 沿海公路、蓝天大海
4. **咖啡馆午后** - 温馨的室内场景、阳光洒落
5. **雪夜漫步** - 冬日雪景、温暖氛围
6. **书店时光** - 书架、阅读的场景

## 优化建议

- 使用工具（如 TinyPNG）压缩图片
- 确保图片方向正确
- 避免过度曝光或过暗的照片
- 建议色调偏向电影感（稍微降低饱和度）

## 如何替换占位符

在 `src/components/gallery/Gallery.tsx` 中取消注释 Image 组件部分即可启用真实图片加载。
