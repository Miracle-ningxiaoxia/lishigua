# 地球贴图资源

## 智能分级加载

系统会根据可用的贴图自动选择最佳渲染方案：

- **完整版**：需要全部 4 个贴图（最佳效果）
- **基础版**：只需 2 个基础贴图（推荐）
- **简化版**：无需贴图（纯色材质）

## 贴图文件说明

### 🔴 必需贴图（基础版）

#### 1. `earth-day.jpg` - 日景贴图
- **描述**：地球白天的真实纹理，包含陆地、海洋、云层等
- **必需**：✅ 是
- **推荐分辨率**：2048x1024（或 4096x2048）
- **下载地址**：https://www.solarsystemscope.com/textures/

#### 2. `earth-night.jpg` - 夜景灯光贴图
- **描述**：地球夜晚的城市灯光效果
- **必需**：✅ 是
- **推荐分辨率**：2048x1024（或 4096x2048）
- **下载地址**：https://www.solarsystemscope.com/textures/

> 💡 **提示**：只下载这两个文件就能看到日夜切换和城市灯光效果！

### 🟡 可选贴图（完整版）

#### 3. `earth-normal.jpg` - 法线贴图
- **描述**：用于增强地球表面的立体细节（山脉、海沟等）
- **必需**：⚠️ 可选（增强效果）
- **推荐分辨率**：2048x1024
- **下载地址**：https://www.solarsystemscope.com/textures/

#### 4. `earth-specular.jpg` - 高光贴图
- **描述**：控制地球表面的反光效果（海洋反光强，陆地反光弱）
- **必需**：⚠️ 可选（增强效果）
- **推荐分辨率**：2048x1024
- **下载地址**：https://www.solarsystemscope.com/textures/

## 使用说明

1. 下载上述贴图文件
2. 将它们放置在 `public/textures/` 目录下
3. 确保文件名与代码中引用的名称一致
4. 重启开发服务器即可看到效果
