# Intro Page 更新说明

## 📋 更新概述

Intro 页面已从气球互动页面更新为优雅的标题展示页。

## 🎨 新设计特点

### 视觉元素

1. **全屏视频背景**
   - 播放 `public/hero-bg.mp4`
   - 40% 不透明度，带渐变遮罩
   - 自动播放、循环、静音

2. **主标题 - 拾光纪**
   - 超大字号：8xl-12rem（响应式）
   - 轻薄字重，宽字间距
   - 发光效果（模糊叠加层）
   - 平滑缩放入场动画

3. **英文点缀**
   - 顶部：`Welcome to`（小字，上移入场）
   - 副标题：`Moments Collected`（等宽字体，大写）
   - 底部：`Click to Begin Your Journey`（最小字号）

4. **艺术装饰**
   - 水平装饰线（渐变，缩放入场）
   - 诗意文字：`时光的碎片，友谊的印记`（手写字体）
   - 双语排版：中文+英文
   - 20个浮动粒子（随机位置，上下浮动）

### 交互元素

**Enter 按钮**
- 玻璃态设计（毛玻璃效果）
- 圆角按钮，带图标（ArrowRight）
- 悬停效果：
  - 放大（1.05倍）
  - 发光阴影
  - 图标右移
  - 渐变光泽扫过
- 点击后进入成员滚动页面
- 点击时启动背景音乐

## 🔧 技术实现

### 新文件
- `src/components/intro/IntroTitle.tsx` - 新的标题展示页组件

### 修改文件
- `src/components/intro/IntroOrchestrator.tsx` - 使用新组件替代气球组件

### 保留文件
- `src/components/intro/IntroBubble.tsx` - 保留但不再使用

## 📦 组件结构

```typescript
IntroTitle
├── Background Video
│   ├── <video> 元素
│   └── 渐变遮罩
├── Content Container
│   ├── 装饰文字："Welcome to"
│   ├── 主标题："拾光纪"（带发光效果）
│   ├── 英文副标题："Moments Collected"
│   ├── 装饰线
│   ├── 诗意文字（中英双语）
│   ├── Enter 按钮
│   └── 底部提示文字
└── 浮动粒子层
```

## 🎭 动画时序

```
0.0s  页面加载
0.3s  → "Welcome to" 淡入上移
0.5s  → 主标题 "拾光纪" 缩放淡入
0.8s  → 英文副标题淡入上移
1.0s  → 装饰线缩放展开
1.2s  → 诗意文字淡入上移
1.5s  → Enter 按钮淡入上移
2.0s  → 底部提示文字淡入
持续  → 浮动粒子动画
持续  → 按钮光泽扫过动画
```

## 🎨 样式细节

### 字体
- 主标题：默认字体，light weight
- 英文/代码：Monospace（等宽字体）
- 诗意文字：Dancing Script（手写字体）

### 颜色
- 白色为主
- 不同透明度：
  - 主标题：90%
  - 副标题：60%
  - 诗意文字：70%
  - 提示文字：20%-40%

### 间距
- 字母间距：0.15em - 0.5em（非常宽）
- 垂直间距：合理分布，视觉平衡

## 🚀 用户流程

### 之前
```
访问 /intro
  ↓
气球页面（需要点击气球）
  ↓
气球爆炸动画
  ↓
成员滚动页面
```

### 现在
```
访问 /intro
  ↓
标题展示页（拾光纪）
  ↓
点击 Enter 按钮
  ↓
启动音乐 + 淡出过渡
  ↓
成员滚动页面
```

## 💡 设计理念

1. **简约优雅**
   - 大面积留白
   - 简洁的排版
   - 克制的动画

2. **沉浸感**
   - 全屏视频背景
   - 深色主题
   - 柔和的动画

3. **层次感**
   - 视频底层
   - 内容中层
   - 粒子顶层

4. **引导性**
   - 明确的 Enter 按钮
   - 视觉焦点集中
   - 清晰的行动号召

## 📱 响应式设计

### 移动端
- 字号调整：8xl
- 保持中心对齐
- 按钮大小适配
- 粒子数量不变

### 平板
- 字号：9xl
- 间距微调

### 桌面
- 字号：12rem（最大）
- 最佳视觉体验

## 🎵 音乐触发

- 点击 Enter 按钮时触发
- 调用 `onMusicStart()` 回调
- 背景音乐淡入播放
- 音乐持续到其他页面

## ⚡ 性能优化

1. **视频优化**
   - 使用合适的视频格式
   - 适当压缩
   - 懒加载（考虑添加）

2. **动画优化**
   - 使用 CSS transforms（硬件加速）
   - Framer Motion 优化
   - 避免 layout 抖动

3. **粒子优化**
   - 仅 20 个粒子
   - 简单的上下浮动
   - 使用 absolute 定位
